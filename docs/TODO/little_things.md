


- Si se ingesta un bundle con un geo-strategic-actor que se da de alta. Si luego se quiere ingestar otro JSON ese actor no está disponible en el campo "Target Actor". Sino que se tiene que ir a otra sección y volver de nuevo a "Data Ingestion" para que aparezca.

- Una vez se ingesta el JSON de un bundle el "Select existing actor" se queda informado con el último valor seleccionado. No se pone en blanco para que el usuario seleccione otro actor.
